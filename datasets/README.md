# Folder structure

The folder structure should follow below:

```shell
tree -L 3
.
├── CCS
│   ├── sequence_name
│       ├── basler_0/
│       ├── prophesee_0/
│       └── homography.txt
│           ...
└── README.md  # this readme 
```

Please download datasets accordingly.
